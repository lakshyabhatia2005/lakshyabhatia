## Projects

### [📣 Marketing Campaign Performance Analysis](https://github.com/lakshyabhatia10/marketing-campaign-analysis)
Analyzed $252M in marketing spend across campaigns to uncover a critical problem — strong ROI (275%) but a 20% conversion rate. Applied Logistic Regression for conversion prediction and K-Means for customer segmentation. Delivered a Power BI dashboard with actionable budget reallocation recommendations.
`Python` `Power BI` `Logistic Regression` `K-Means` `EDA`

---

### [📱 Social Media Engagement Analysis for Influencer Marketing](https://github.com/lakshyabhatia10/social-media-engagement-analysis)
Investigated how platform, posting time, sentiment, and campaign phase drive engagement rate. Found that midday posts outperform others and neutral/negative content generates higher interaction — counterintuitive findings backed by data.
`Python` `Pandas` `Seaborn` `Matplotlib` `EDA`

---

### [🏥 Hospital Patient Management System](https://github.com/lakshyabhatia10/hospital-patient-management-sql)
Designed a normalized relational database with 9 tables managing patients, staff, appointments, billing, and room allocation. Implemented triggers for audit trails, stored procedures for data entry, and indexes for query performance.
`MySQL` `SQL` `Database Design` `Triggers` `Stored Procedures`

---

### [📅 Event Planner: Registration & Analysis System](https://github.com/lakshyabhatia10/event-planner-python)
Built an end-to-end event management system using Python OOP. Integrated Groq LLM API for AI-driven crowd prediction and theme suggestions. Exported structured reports to Excel automatically.
`Python` `Pandas` `OOP` `Groq AI API` `Excel`

---

### [🏏 ODI Cricket Insights](https://github.com/lakshyabhatia10/odi-cricket-insights)
Performed data analysis on 1,500+ ODI cricketers to objectively rank players and teams. Used quartile analysis, composite scoring, and pivot tables to surface insights on batting dominance, award leaders, and win/loss patterns.
`Excel` `Pivot Tables` `Statistical Analysis` `Sports Analytics`
